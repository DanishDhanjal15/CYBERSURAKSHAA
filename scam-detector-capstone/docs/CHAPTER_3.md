# Chapter 3: Analysis and Planning

## 3.1 Introduction

This chapter is dedicated to the analysis and planning of the suggested AI-based system for the analysis of cybersecurity threats and malicious social media messages. Analysis and planning are critical steps in the development of any software project, as they enable a proper understanding of the requirements, setting the goals, and the proper management of the project development. By proper planning, the project will be implemented effectively, on time, and according to the expected goals.

In this project, the analysis phase includes identifying the requirements for the integration of technical domain-age validation with deep multilingual semantic logic. In addition, the analysis phase includes understanding the interactions between the different components of the system. It also includes understanding the requirements of the users, the tools required for the project, and the workflow of the system.

On the other hand, project planning is the structuring of the project in different modules, including data collection, technical preprocessing, ensemble text analytics, and visualization. It is also the allocation of resources, the estimation of the project timeline, and the distribution of the project for the smooth execution of the different phases.

This chapter will further explain the project planning strategy and the project scheduling process, which is crucial for the successful implementation of the project.

---

## 3.2 Project Planning

The process of project planning includes defining the objectives, scope, and resources to be used for the successful development of the system. In this project, the main objective is to design a system that uses AI to integrate unstructured text analytics with real-time domain infrastructure checking.

The tools and technologies to be used for this project are identified in the planning phase. This includes using Python, FastAPI, React, Tailwind CSS, and advanced ML/DL models (XGBoost, XLM-RoBERTa). Additionally, it includes identifying the data sources and libraries to be used for this project, such as the `python-whois` library for infrastructure tracking.

The project has been divided into different modules. This includes collecting messaging data, preprocessing raw text and URLs, performing dual-engine semantic analysis, and visualizing the final threat metrics. This modular approach to project development facilitates easier debugging. Hence, it ensures that all stages of project development are performed efficiently.

## 3.3 Scheduling 

Scheduling is a significant aspect of project planning that ensures all project activities are completed within a particular period. Scheduling is a process that divides a project into various phases, with a specific time duration for each project activity to ensure a smooth workflow. A well-defined schedule for a project enables effective monitoring, efficient use of resources, and minimization of delays during project development.

In this project, the development process has been divided into various phases. The first phase is requirement analysis, where a problem statement regarding social media scams and objectives are defined. The next phase is data collection, where regional scam datasets and raw text are collected and translated. The third phase of project development is data and URL preprocessing. 

After that, the dual-engine analytical models are implemented to classify the messages into safe, suspicious, or high-risk scam categories. Next is the system development phase, where the dashboard is created using React and visually styled with a Traffic Light paradigm. After the completion of the above phases, the system is tested for its accuracy and integrated with the Python backend. Finally, the project is deployed.

The phases of the project are scheduled in such a manner that each phase is completed within a specified period of time in a systematic way. It also enables tracking the progress of the project with the necessary changes to ensure the completion of the project in a successful manner.
